# 🐍 Python para Análise de Dados e Ciência de Dados

Bem-vindo à seção de projetos desenvolvidos em **Python**. Este espaço é dedicado à documentação de scripts, notebooks e aplicações voltadas para o ciclo completo de dados: desde a extração e limpeza até a modelagem preditiva e automação.

---

## 🛠️ Stack Tecnológica e Bibliotecas

Para os projetos contidos aqui, utilizo o ecossistema científico e analítico do Python:

* **Pandas:** Manipulação de DataFrames, limpeza e transformação de dados estruturados.
* **NumPy:** Operações matemáticas, álgebra linear e computação vetorial de alta performance.
* **Matplotlib & Seaborn:** Criação de visualizações estáticas, informativas e estatísticas para EDA.
* **Scikit-Learn:** Implementação de algoritmos de Machine Learning (Regressão, Classificação e Clusterização).
* **Requests / BeautifulSoup:** Técnicas de Web Scraping e consumo de APIs para coleta de dados.

---

## 📂 Organização dos Projetos

### 1. Análise Exploratória de Dados (EDA)
Projetos focados em entender o comportamento dos dados antes da modelagem.
* **Técnicas:** Tratamento de dados ausentes, análise de correlação, identificação de *outliers* e distribuições.
* **Objetivo:** Gerar insights acionáveis e preparar os dados para o negócio ou para modelos de ML.

### 2. Engenharia de Dados & ETL com Python
Scripts focados em automação e movimentação de dados.
* **Técnicas:** Limpeza automatizada, padronização de tipos de dados e exportação para bancos de dados SQL ou arquivos otimizados (Parquet).
* **Objetivo:** Reduzir o trabalho manual e garantir a qualidade da informação na origem.

### 3. Machine Learning e Ciência de Dados
Aplicação de modelos estatísticos e preditivos para solução de problemas complexos.
* **Técnicas:** Engenharia de Atributos (*Feature Engineering*), treinamento de modelos, ajuste de hiperparâmetros e validação de métricas (RMSE, MAE, Acurácia).
