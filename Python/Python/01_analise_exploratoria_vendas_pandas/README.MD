# 📊 Projeto 01: Manipulação e Análise de Vendas com Pandas

Este é o projeto inicial da trilha de Python para Dados. O objetivo foi realizar uma análise exploratória básica (EDA) de um dataset de vendas, demonstrando competência no uso da biblioteca Pandas para tratamento de informações.

## 🎯 Objetivos do Projeto
* **Estruturação de Dados:** Criação de DataFrames a partir de dicionários.
* **Data Cleaning:** Conversão de tipos de dados (Datas) para manipulação temporal correta.
* **Cálculo de Indicadores:** Geração de colunas calculadas para métricas de faturamento (KPIs).
* **Análise Inicial:** Uso de métodos como `.head()` e cálculos aritméticos em colunas.

## 🛠️ Tecnologias Utilizadas
* **Python 3.x**
* **Pandas:** Biblioteca para manipulação e análise de dados.
* **Matplotlib:** Configurações iniciais para visualização.

## 📂 O que foi desenvolvido?
O código simula o cenário de uma loja, onde processamos os pedidos de diferentes periféricos (Notebook, Mouse, Monitor, etc.). 
- Foi criada a métrica **Faturamento Total** (`Quantidade * Preço_Unitário`), permitindo visualizar o valor gerado por cada transação individualmente.
